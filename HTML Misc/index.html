Que es un meta? Meta viene de metadato. Que es un metadato¿? Es un dato que describe a otro dato.
Cuando se inserta un tag de meta cosas como explicarle al navegador como reconocer ciertos carácteres
(es decir, le describimos datos al navegador) para que este despues los ilustre en la página de manera
correcta. Suelen estar compuestos en el siguiente orden:
meta | name="tipo de metadato" y content="contenido debido según función del metadato">.
Es mas fácil entenderlo en la practica que en la teoría.

Charset por ejemplo viene de Character set, del inglés ajustar carácter.
Y que tipos de caracter queremos describirle exactamente? los de UTF-8. De esta manera el navegador
reconocerá acentos, simbolos y otros caracteres que antes no. Resultando en letras raras que dan mal
aspecto y nula utilidad.

Sirve para indexar o no la página, es decir filtrar
la busqueda mediante los datos anteriormente insertos en el meta keyword y meta description.
Si elegimos noindex, entonces por mas que nuestro taller mecanico tenga keywords 'autos reparación balanceo
ejes' y se busque con dichas palabras, el buscador no las tendrá en cuenta. Esta opción es para hacer lo
contrario a facilitar la busqueda de la pagina a los usuarios. Una función para paginas 'secretas'.

Para vincular una imagen se usa
img src="dirección del sitio/carpeta que contiene la imagen" alt="Texto que aparecera en caso de que la imagen no se pueda cargar">


Ejemplo:<img src=Tuvieja.jpg alt= Foto de tuvieja>


El Alt= tambien es muy importante para el buscador como palabra clave, y para aquellos que tienen
un buscador para ciegos, dictará lo que está en el alt para comprender la imagen. Ahora, también se
le puede agregar un atributo 'title' después del Alt. Este no es obligatorio, lo que hace es manifestar
en un pequeño rectangulo de texto al lado del mouse cuando este señala la imagen. El texto del recuadro
es lo escrito en 'title=...' .


Para vincular un ícono se usa

Para vincular un video se usa

<video> y </video>
Dentro de estos tags se encuentra un src="dirección del sitio/carpeta que contiene el video" y al lado de esto
añadimos el atributo 'controls' que permite decirle al buscador que controles de video usar
Los controles de video son esos que nos ayudan para interactuar con el archivo sea para pausarlo,
rebobinarlo, subir o bajar el volumen y poner el contenido en pantalla completa entre otros.


Para vincular un audio:
En vez de video el comando es <audio></audio>. Se usa el 'src' y siempre que el

Para insertar una entrada de información (input) se usa el tag <input type="text">
Por defecto cuando usamos el tag, ya viene configurada la entrada para insertar únicamente texto
Existen muchos tipos de input, estos son los mas usados;

    • *<input type="text">* Tipea solo texto
    • *<input type="number">* Solo números
    • *<input type="email">* Solo conjunto de caracteres
    de combinacion ***@***.***
    • *<input type=password>* Solo para poner contraseñas
    • *<input type=submit>* Botón utilizado para enviar los datos al servidor que fueron registrados en las entradas previas.

Nota: Lo que se haya escrito dentro de los tags input es reconocido como un valor o 'value' en inglés. Esto se puede poner en Html pero es algo que normalmente usamos en JavaScript.
Ejemplo: <input type="text" value="Que onda Juan."> Entonces cuando se abra la página lo que
figurará dentro de la casilla es el valor 'Que onda Juan'.

Ahora bien, para que los tags de entrada anteriores respondan deben estar encerrados entre un tag llamado form que quiere decir 'formulario', y se escribe de la siguiente manera;
<form action="" method="post"> </form>
Funciona incluso escribiendoló así;
<form method="post"> </form> Esta sería la manera mas simple de escribirlo, existen otros 'method' y por lo visto 'action='' también desempeña un papel el cual aún desconozco.

Hacer un atajo en una misma pagina se requiere del comando href, pero antes alarguemos la
pagina en secciones para poder hacer los atajos.

Una vez extendido el alto de la pagina, separaremos por secciones las cuales serán los párrafos.

1- Se coloca 'id=' y el nombre que le asignamos al vinculo que hara el salto dentro de la página.


2- Se coloca un a href= que comienza con # y el nombre elegido del vinculo.


3- Si quisiera volver al principio de la pagina, inserto el siguiente texto a la altura que quisiera
el 'botón' del comienzo: a href=#. Con solo poner numeral la accion sera saltar al comienzo. Después
completamos con el nombre que queremos indicar a dicho botón.
